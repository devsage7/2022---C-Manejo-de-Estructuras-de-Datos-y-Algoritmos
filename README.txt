Este es un proyecto académico hecho para la materia "Estructuras de Datos y Algoritmos".
Decidí que mostrar las tres entregas separadas era una buena forma de que se vea el progreso 
y la correción de errores que fuí haciendo a lo largo de ese cuatrimestre.

de las estructuras de datos más usadas y un decente conocimiento sobre algoritmos.

Cabe destacar que la materia tenía muchísima teoría, esto es solo una parte de lo que aprendí.
